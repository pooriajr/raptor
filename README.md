# Raptor (React)

React + TypeScript + Vite implementation of **Raptor**, an asymmetric 2‑player board game.

The app is a single-screen “pass-and-play” flow (scientist selects first, then raptor), driven by a reducer-based phase/state machine.

## Development

```bash
npm install
npm run dev
```

## Useful Commands

```bash
npm run build
npm run lint
npm test
npm run test:ui
```
